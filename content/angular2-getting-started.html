Quick Start

To install and use the Angular CLI:

npm install -g angular-cli
npm update -g angular-cli    // To update angular-cli
ng new project_name
cd project_name
 
ng serve

Directory structure

config - Environment & testing
dist - Contains the compiled application. Nothing for developers to change here
e2e - End to End testing
public - Directory which contains all our stylesheets, script libraries (External scripts, JQuery, etc), images
src - All our work goes here
src/app - Contains all the components, service, etc
src/app/app.component.spec.ts - Used in testing
typings -
angular-cli.json -
angular-cli-build.json -

Dev commands

ng g component todos - Creates the skeleton framework in src/apps for the new component todos

Quick Start - Material

Installing Angular 2 Material Design Packages
npm install --save @angular2-material/core @angular2-material/card @angular2-material/icon @angular2-material/toolbar
-- The core module is necessary for all Angular 2 material design
Install HammerJS
npm install --save hammerjs
-- Hammer add support for touch gestures and is used by the Icon Material components
npm install --save @types/hammerjs
//typings install dt~hammerjs --save --global
-- Add the typings of hammerjs
SystemJS configuration
Add the Angular 2 Material NPM packages to systemjs.config.js by using the UMD (Universal Module Definition) files
map: { // our app is within the app folder app: 'app', // angular bundles '@angular/core': 'npm:@angular/core/bundles/core.umd.js', '@angular/common': 'npm:@angular/common/bundles/common.umd.js', '@angular/compiler': 'npm:@angular/compiler/bundles /compiler.umd.js', '@angular/platform-browser': 'npm:@angular/platform-browser/bundles/platform-browser.umd.js', '@angular/platform-browser-dynamic': 'npm:@angular/platform-browser-dynamic/bundles/platform-browser- dynamic.umd.js', '@angular/http': 'npm:@angular/http/bundles/http.umd.js', '@angular/router': 'npm:@angular/router/bundles/router.umd.js', '@angular/forms': 'npm:@angular/forms/bundles/forms.umd.js', // other libraries 'rxjs': 'npm:rxjs', 'angular2-in-memory-web-api': 'npm:angular2-in-memory-web-api', '@angular2-material/core': 'npm:@angular2-material/core/core.umd.js', '@angular2-material/card': 'npm:@angular2-material/card/card.umd.js', '@angular2-material/button': 'npm:@angular2-material/button/button.umd.js', '@angular2-material/icon': 'npm:@angular2-material/icon/icon.umd.js', },
