Imp Notes

  1. I have pointed the application services to use node ports. As per Kubernetes logic, the service listens at that port from all nodes of the kubernetes cluster.
  2. Added DNS Host A multiple entries with same FQDN name and all nodes IPs
  3. Used nginx for reverse proxy
  4. Used NFS share (which is being backed up using vDP - vSphere Data Protection) for application data so that we don't loose any configuration.
